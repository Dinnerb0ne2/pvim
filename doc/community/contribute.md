# Contributing

感谢你关注 PVIM。

## 贡献方式

1. 提交 Issue：描述问题、复现步骤和期望行为
2. 提交 PR：聚焦单一目标，附带必要说明
3. 改文档：欢迎补充教程、FAQ 与架构说明

## 代码约定

- 纯 Python 标准库实现（不引入第三方依赖）
- 模块化拆分，避免超大函数持续膨胀
- 错误要可见，禁止静默吞错

## 提交前检查

- `python -m compileall src pvim.py`
- 关键功能手动回归（编辑、撤销、保存、会话恢复）

## 许可证

提交到本项目的代码默认以 **GPLv3.0-or-later** 许可发布。
