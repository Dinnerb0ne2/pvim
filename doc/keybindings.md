# 快捷键与操作

## 全局

- `Ctrl+S`：保存
- `Ctrl+Q` / `Ctrl+C`：退出
- `F1`：打开快捷键提示面板（可配置）

## 模式切换

- `i`：进入 INSERT
- `Esc`：回到 NORMAL
- `:`：进入命令行
- `V`：进入 VISUAL LINE

## 编辑能力

- `Ctrl+/`：一键注释/取消注释
- `Ctrl+D`：向下添加多光标
- `Ctrl+U`：清空多光标
- `Tab` / `Shift+Tab`：缩进 / 反缩进
- `Ctrl+Left` / `Ctrl+Right`：按单词移动

## 工具能力

- `Ctrl+F`：快速查找（预填 `:find `）
- `Ctrl+G`：快速替换（预填 `:replace `）
- `Ctrl+P`：模糊搜索器
- `Ctrl+R`：重命名命令（基于光标下符号）
- `F4`：切换侧边栏
- `F8`：代码风格归一化

## 提示面板

按 `F1` 后可进入快捷键提示页：

- `Up/Down`：滚动
- `Esc/Enter/F1`：关闭
