# 快捷键与交互

## 通用

- `Ctrl+S` 保存
- `Ctrl+Q` / `Ctrl+C` 退出
- `F1` 快捷键提示面板

## 编辑

- `Ctrl+/` 注释切换
- `Ctrl+D` 添加多光标
- `Ctrl+U` 清空多光标
- `Ctrl+Left` / `Ctrl+Right` 按单词移动
- `Tab` / `Shift+Tab` 缩进/反缩进

## 导航与工具

- `Ctrl+P` 模糊搜索
- `Ctrl+F` 快速查找
- `Ctrl+G` 快速替换
- `Ctrl+R` 重命名入口
- `F4` 侧边栏开关
- `F8` 风格归一化

## 浮动列表交互（统一组件）

- `Up/Down` 移动选项
- `Enter` 选择当前项
- `Esc` 关闭弹层
